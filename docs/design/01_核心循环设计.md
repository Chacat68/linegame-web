# 星际贸易商 - 核心游戏循环设计

## 系统概述

核心游戏循环是玩家在游戏中反复体验的核心机制，良好的循环设计能够保持玩家的持续参与度和成就感。《星际贸易商》采用三层嵌套循环设计，从短期的单次交易到长期的商业帝国建设，为玩家提供不同时间尺度的游戏目标和反馈。

## 三层核心循环架构

### 微循环（单次交易，5-10分钟）

**循环流程：**
1. **发现需求** - 查看市场信息，识别价格差异
2. **购买商品** - 在低价星球购买货物
3. **星际航行** - 前往目标星球
4. **卖出商品** - 在高价星球出售货物
5. **获得利润** - 结算收益，获取经验

**反馈机制：**
- **即时利润显示**：交易完成时弹出利润结算面板
  - 显示买入价、卖出价、净利润
  - 利润率以颜色区分（绿色：>20%，黄色：10-20%，白色：<10%）
  - 播放"叮"音效 + 金币动画
  
- **经验获取**：
  - 每次成功交易获得经验值
  - 经验值 = 基础经验 × (1 + 利润率 × 0.5)
  - 进度条实时显示升级进度
  
- **技能提升提示**：
  - 谈判技能经验累积
  - 达到技能阈值时显示技能升级通知

**心流设计：**
- 决策点清晰：市场界面高亮推荐商品
- 操作流畅：一键购买、一键航行
- 风险适中：5-15%的价格波动，偶尔遭遇小事件
- 奖励及时：交易后立即看到收益

---

### 中循环（贸易路线经营，30-60分钟）

**循环流程：**
1. **规划路线** - 分析3-5个星球的商品供需
2. **多次交易** - 沿路线完成4-6次交易
3. **升级飞船** - 使用积累资金购买更大货舱或更快引擎
4. **解锁新星球** - 达到声望要求，开启新贸易区域
5. **优化路线** - 整合新星球，提升路线效率

**反馈机制：**
- **等级提升**：
  - 玩家等级每5级解锁新功能
  - 升级时全屏特效 + 音效
  - 显示解锁内容清单（新飞船、新星球、新功能）
  
- **声望变化**：
  - 每次交易后显示声望增长 (+5, +10, +20)
  - 声望达到阈值时解锁星球特权（当前实现，见 `js/data/playerLevels.js`）：
    - 小有名气（50声望）：交易折扣 2%
    - 声名远扬（150声望）：交易折扣 5%
    - 商业精英（400声望）：交易折扣 8%
    - 银河传奇（800声望）：交易折扣 12%
  
- **新区域解锁**：
  - 星图上新星球以"闪烁"效果出现
  - 弹窗介绍新星球特性和商品
  - 提示推荐贸易路线

**心流设计：**
- 策略深度提升：需要考虑多个星球的价格波动
- 成长感明显：飞船升级带来可见的效率提升
- 探索奖励：新星球带来新商品和新机会
- 风险管理：需要平衡投资与储备资金

---

### 宏循环（商业帝国建设，数小时~数十小时）

**循环流程：**
1. **建立贸易网** - 在多个星球建立贸易站
2. **影响经济** - 通过大宗交易影响星球价格
3. **派系抉择** - 选择支持的政治/商业派系
4. **剧情推进** - 完成主线任务，改变银河格局
5. **达成结局** - 根据选择触发不同结局

**反馈机制：**
- **称号与成就**：
  - 财富里程碑成就：
    - "小有积蓄"（100万SC）
    - "商业新星"（500万SC）
    - "星际大亨"（2000万SC）
    - "银河首富"（1亿SC）
  - 特殊成就：完美交易、探索家、外交官、走私之王
  
- **世界观变化**：
  - 玩家的选择影响星球状态
  - 新闻系统报道玩家的重大交易
  - NPC对话内容根据玩家声望变化
  
- **结局分支**：
  - 和平商人结局：通过贸易促进银河和平
  - 商业帝国结局：控制主要贸易路线
  - 走私大王结局：主导黑市交易
  - 革命家结局：推翻腐败政权
  - 探索者结局：发现古代文明遗迹

**心流设计：**
- 长期目标清晰：建立商业帝国的愿景
- 选择有意义：派系决策影响游戏世界
- 剧情驱动参与：主线任务提供叙事动力
- 个性化体验：不同玩法路线导向不同结局

## 循环间的衔接机制

### 微循环 → 中循环的过渡
**触发条件：**
- 完成5-10次单次交易
- 玩家等级达到3级
- 积累资金超过初始资金的2倍

**衔接机制：**
- **教程触发**："星图导航"教程自动弹出
- **系统解锁**：路线规划功能激活
- **引导任务**："建立你的第一条贸易路线"任务出现
- **奖励激励**：完成首条路线额外奖励10%利润

### 中循环 → 宏循环的过渡
**触发条件：**
- 玩家等级达到10级
- 拥有至少2艘飞船
- 在3个以上星球达到"银牌商人"声望

**衔接机制：**
- **剧情触发**：主线剧情"银河商会的邀请"开启
- **系统解锁**：贸易站建设、派系系统激活
- **战略层开放**：可查看银河经济概览图
- **长期目标展示**：成就系统完整解锁

## 心流节奏设计

### 紧张-放松曲线

**紧张阶段（活跃操作）：**
- **持续时间**：15-30分钟
- **内容**：
  - 连续交易（微循环）
  - 处理随机事件（海盗、价格暴跌）
  - 飞船战斗或逃跑
  - 时间紧迫的限时任务
- **设计要素**：
  - 快节奏音乐
  - 动态事件触发
  - 倒计时机制
  - 风险与收益并存

**放松阶段（休整规划）：**
- **持续时间**：5-10分钟
- **内容**：
  - 查看市场数据
  - 规划下一条路线
  - 飞船维护和升级
  - 查看成就和收藏
- **设计要素**：
  - 舒缓背景音乐
  - 无时间压力
  - UI界面操作
  - 战略思考空间

### 节奏控制机制

**玩家主导节奏：**
- 允许玩家选择是否接受高风险任务
- 随时可返回安全星球进行整备
- 无强制战斗，可选择逃跑

**系统引导节奏：**
- 随机事件频率随游戏时长动态调整
  - 前15分钟：低频率（每10分钟1次）
  - 15-60分钟：中频率（每5分钟1次）
  - 60分钟后：恢复低频率，但事件更复杂
- 价格波动周期：每20-30分钟一次市场刷新

## 正向反馈循环设计

### 经济雪球效应
```
更多资金 → 购买更大飞船 → 单次运输更多货物 → 更高利润 → 更多资金
```

**平衡机制：**
- 飞船运营成本随价值增长
- 高价值货物风险更高（海盗针对）
- 市场饱和机制（大量卖出导致价格下跌）

### 技能成长循环
```
成功交易 → 获得经验 → 技能提升 → 更好交易条件 → 更成功交易
```

**平衡机制：**
- 技能升级成本递增
- 后期技能提升需要特殊任务解锁
- 技能上限（每项技能最高等级10）

### 声望增长循环
```
完成任务 → 声望提升 → 解锁特权 → 接取更高级任务 → 更多声望
```

**平衡机制：**
- 不同派系声望互斥（支持A派系会降低B派系声望）
- 声望衰减机制（长期不交易会缓慢下降）
- 声望恢复需要特殊任务

## 反向压力机制

### 经济压力
- **日常开支**：船员薪资、燃料、维护费
- **贷款利息**：如果选择贷款发展
- **通货膨胀**：随着玩家资产增长，物价上调

### 竞争压力
- **NPC竞争对手**：抢占相同贸易路线
- **市场动态**：玩家行为影响价格，需要不断寻找新机会

### 时间压力
- **限时任务**：高额奖励但必须在规定时间内完成
- **季节性商品**：特定时期价格最高
- **事件倒计时**：某些事件必须快速决策

---

## 循环失败与恢复机制

### 失败容错设计
**破产保护：**
- 玩家资金归零时不会Game Over
- 触发"破产保护"任务
  - 银河商会提供低息贷款（10,000 SC）
  - 强制接取低风险运输任务还债
  - 完成3次任务后恢复正常游戏

**飞船损毁：**
- 飞船被摧毁时保留50%货物价值作为保险赔偿
- 降级到最基础飞船
- 保留所有技能和声望

### 恢复激励
- 失败后首次盈利获得"东山再起"成就
- 恢复期间随机事件概率降低50%
- 临时增加基础任务奖励20%

## 循环深度的渐进解锁

### 第一阶段（0-2小时）
**开放系统：**
- 基础贸易（微循环）
- 5个初始星球
- 8种基础商品

**隐藏系统：**
- 路线规划、飞船升级、随机事件

### 第二阶段（2-10小时）
**开放系统：**
- 路线规划（中循环）
- 飞船升级系统
- 完整随机事件表
- 10个额外星球

**隐藏系统：**
- 派系系统、贸易站、主线剧情

### 第三阶段（10小时+）
**开放系统：**
- 派系系统（宏循环）
- 贸易站建设
- 主线剧情
- 舰队管理
- 黑市与走私

## 玩家留存机制

### 短期留存（单次游戏时长：30-60分钟）
- **即时目标**：完成一条完整贸易路线
- **进度可见**：经验条、资金增长
- **快速奖励**：每次交易的利润反馈

### 中期留存（回访周期：1-3天）
- **每日任务**：每日刷新的高奖励任务
- **限时事件**：周末特殊事件（双倍声望、稀有商品）
- **成长感**：飞船升级、技能提升

### 长期留存（游戏生命周期：数周~数月）
- **长期目标**：建立商业帝国
- **剧情驱动**：主线任务揭示世界观
- **成就收集**：50+成就系统
- **多周目价值**：不同派系路线、不同结局

---

## 数据埋点与循环优化

### 关键指标监测
- **微循环完成率**：有多少玩家完成首次交易
- **中循环进入率**：有多少玩家开始规划路线
- **宏循环参与率**：有多少玩家触发主线剧情

### 优化调整方向
- 如果微循环完成率<80%，简化新手引导
- 如果中循环进入率<50%，增加路线规划的奖励激励
- 如果宏循环参与率<30%，提前触发剧情引导

## 设计原则总结

1. **短期回报，长期深度**：微循环提供即时满足，宏循环提供持久目标
2. **渐进复杂度**：系统逐步解锁，避免新手信息过载
3. **玩家主导节奏**：允许玩家选择快节奏或慢节奏玩法
4. **有意义的选择**：每个决策都应该有可见的影响
5. **失败友好**：提供恢复机制，鼓励冒险
6. **正负反馈平衡**：成长与压力并存，保持挑战性
