# 星际贸易商 - 随机事件表

## 系统概述

随机事件是《星际贸易商》增加游戏趣味性和不确定性的重要机制。本文档详细定义了游戏中的所有随机事件，包括触发条件、选项、结果和奖惩机制。事件覆盖航行、市场、船员、探索、政治、自然、贸易、社交、技术等多个类别。

---

## 事件触发机制

### 触发概率公式

**基础触发率：**
```
每小时事件触发概率 = 基础概率(10%) × 区域系数 × 运气系数

区域系数：
- 安全区：0.5（低频）
- 中立区：1.0（标准）
- 危险区：2.0（高频）
- 无法区：3.0（极高频）

运气系数（可选技能）：
- 无运气技能：1.0
- 运气+1：1.2
- 运气+5：2.0
```

### 事件冷却时间

```
全局冷却：同一事件不会在冷却时间内重复触发
特定事件冷却：
- 普通事件：10分钟
- 稀有事件：30分钟
- 传奇事件：60分钟
```

**选项A：接受合同**
- 合同内容：
  - 未来30天内独家供应特定商品
  - 价格优惠15%
  - 每次交易额外奖励5%
  - 违约惩罚：声望-100
- 限制：期间不能与竞争对手交易该商品

**选项B：谈判更好条件（需谈判≥7）**
- 成功率：60%
- 成功结果：
  - 价格优惠提升至20%
  - 奖励提升至8%
- 失败结果：
  - 合同取消
  - 错失机会

**选项C：拒绝**
- 结果：
  - 无任何影响
  - 保持自由交易权

**基于玩家行为的事件：**
