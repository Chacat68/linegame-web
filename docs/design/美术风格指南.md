# 星际贸易商 - 美术风格指南

## 系统概述

本文档定义了《星际贸易商》的完整美术风格规范，确保游戏视觉呈现的一致性和专业性。游戏采用像素艺术风格，结合复古科幻美学和现代UI设计，营造出独特的星际贸易氛围。

本文档扩展了 [11_美术与视觉设计.md](./11_美术与视觉设计.md) 中的内容，提供详细的技术规范。

## 像素分辨率标准

### 角色与飞船精灵

**飞船精灵规格：**
```
尺寸选项：
- 小型飞船/战斗机：32×32 像素
- 中型货船/护卫舰：64×64 像素
- 大型飞船：128×64 像素（宽度优先）
- 巨型母舰：128×128 像素（仅用于特殊场景）

分辨率规则：
- 使用2的幂次方尺寸（便于缩放和优化）
- 主要游戏飞船使用64×64标准
- 保持一致的像素密度（每单位像素数相同）
```

**飞船设计要点：**
```
结构层次：
1. 轮廓层（黑色，1像素）
2. 基础色层（船体主色）
3. 高光层（船体亮部）
4. 细节层（窗口、引擎、标志）

视角：
- 主视图：45度俯视角
- 允许轻微倾斜以展示立体感
- 引擎喷口朝向底部（便于尾焰效果）

示例（64×64货船）：
  ████████████
 ██▓▓▓▓▓▓▓▓██
██▓▓████████▓▓██
██▓███░░░░███▓██  ← 驾驶舱（白色窗口）
██▓██░░░░░░██▓██
██▓▓████████▓▓██
 ████▒▒▒▒████     ← 引擎（暗色）
   ████████
```

**玩家角色/NPC头像：**
```
尺寸：32×32 像素（对话框头像）
风格：半身像，聚焦面部表情
颜色：最多16色（每个角色）

表情变化：
- 中性表情（默认）
- 高兴（交易成功）
- 担忧（遭遇危险）
- 愤怒（被欺骗）
```

### 星球缩略图

**星球图标规格：**
```
尺寸：128×128 像素
用途：星图显示、星球选择界面

绘制层次：
1. 背景（透明或星空）
2. 星球阴影（渐变圆形）
3. 星球表面（纹理细节）
4. 大气层（半透明光晕）

示例分层：
[外层光晕]
  [星球表面纹理]
    [阴影渐变]

```

**星球类型模板：**
```
工业星：

- 纹理：工厂网格、烟雾
- 特征：黄色工业光点
- 大气：薄，灰色

农业星：
- 基础色：绿色 #4CAF50
- 纹理：农田格子、云层
- 特征：蓝色水域

采矿星：
- 基础色：棕橙 #8D6E48
- 纹理：环形坑、矿脉
- 特征：裸露岩石
- 大气：无或极薄

研究星：

娱乐星：

边缘殖民地：

废弃星：
- 基础色：暗红 #5C3A3A
```

### 背景与场景

**星空背景规格：**
```
平铺纹理尺寸：256×256 像素
图层结构：

1. 深空背景层（静态）
   - 深蓝渐变 #0A0E27 → #1A1E47
   - 稀疏星点（白色，随机分布）
   
2. 远景星云层（慢速视差，0.2倍速）
   - 半透明紫/蓝云雾
   - 使用叠加混合模式
   
3. 中景星团层（中速视差，0.5倍速）
   - 较亮的星点
   - 偶尔的彩色星星（红巨星、蓝巨星）
   
4. 近景尘埃层（快速视差，0.8倍速）
   - 暗色尘埃带
   - 增强深度感
```

**港口场景：**
```
尺寸：1920×1080（全屏背景）
风格：像素化的科幻港口

元素构成：
- 背景：远处星球（占画面1/3）
- 中景：太空站结构（金属框架）
- 前景：停靠平台、飞船剪影
- 动态元素：闪烁灯光、飞船来往
```

## 动画规格

### 飞船引擎尾焰

**动画规格：**
```
帧数：3-5帧循环
帧率：12 FPS（每帧83ms）
尺寸：根据飞船大小（通常16×8至32×16）

帧序列示例（小型引擎，16×8）：
帧1:        帧2:        帧3:
  ▓▓          ░▓░         ░░░
 ▓██▓        ▓██▓        ░██░
▓████▓      ░████░      ░████░
 ▓██▓        ▓██▓        ░██░
  ▓▓          ░▓░         ░░░

颜色渐变：
- 核心：白色 #FFFFFF（最热）
- 中层：青色 #00FFFF
- 外层：蓝色 #4169E1
- 尾焰：半透明蓝 #4169E150

循环方式：1→2→3→2→1（乒乓循环）
```

**高级引擎效果：**
```
超空间跃迁效果：
- 引擎逐渐增亮（3帧）
- 飞船拉伸变形（2帧）
- 粒子爆发（1帧）
- 消失淡出（2帧）

总时长：约0.5秒
```

### UI面板动画

**面板展开动画：**
```
时长：0.2秒（200ms）
缓动函数：ease-out（先快后慢）

展开序列：
1. 初始状态：高度0%，透明度0%
2. 中间帧（100ms）：高度60%，透明度80%
3. 结束状态：高度100%，透明度100%

附加效果：
- 边框发光：从无到全
- 内容淡入：延迟50ms开始
```

**按钮悬停动画：**
```
时长：0.1秒（100ms）
效果：
- 边框亮度增加20%
- 背景色变亮10%
- 轻微放大（1.02倍）

点击反馈：
- 按下：缩小至0.98倍
- 释放：回弹至1.0倍
- 总时长：0.15秒
```

### 星球自转

**自转动画规格：**
```
精灵表尺寸：128×128 × 8帧 = 1024×128
帧率：4 FPS（每帧250ms）
总周期：2秒一圈

绘制方法：
1. 绘制基础球体（128×128）
2. 复制并旋转表面纹理45度 × 8次
3. 排列成横向精灵表

优化：
- 小星球（地图视图）使用4帧简化版
- 大星球（详情页）使用完整8帧
```

### 交易成功特效

**特效组合：**
```
1. 金币粒子效果（0-0.5秒）
   - 粒子数量：20-30个
   - 初始位置：利润数字位置
   - 运动：向上抛物线 + 淡出
   - 粒子大小：4×4像素
   - 颜色：金色 #FFD700
   
2. 闪光效果（0.3-0.5秒）
   - 位置：利润数字中心
   - 形状：十字星光
   - 尺寸：从0放大至32×32再缩回
   - 颜色：白色 #FFFFFF
   - 透明度：从100%到0%
   
3. 数字跳动（0.1-0.3秒）
   - 利润数字从0计数到实际值
   - 缓动：ease-out
   - 字体：逐渐放大至1.2倍再回到1.0倍
```

**音效配合：**
```
- 0.0秒："叮"音（高音调）
- 0.1秒：金币碰撞音（低音调）
- 0.3秒：确认音（中音调）
```

## 参考风格

### 《星际牛仔》复古科幻感

**风格要素：**
```
色调：
- 暖色调和冷色调对比
- 爵士乐时代的复古配色
- 霓虹灯效果（但不过度）

氛围：
- 怀旧与未来并存
- "脏兮兮"的真实感（不是全新的太空船）
- 多元文化融合（不同星球的建筑风格差异）
```

**应用到游戏：**
```
- 飞船设计带有磨损感和老旧痕迹
- 港口酒吧使用暖黄色调
- 贸易界面保持冷静的蓝色调
- 爵士风格的背景音乐配合视觉
```

### 《FTL》简洁2D太空风格

**风格要素：**
```
简洁性：
- 清晰的图标设计
- 高对比度
- 功能优先于装饰

信息密度：
- 在有限空间内传达最多信息
- 颜色编码状态
- 直观的符号系统
```

**应用到游戏：**
```
- 飞船状态使用图标+颜色快速识别
- 市场数据使用表格和图表
- 避免过度装饰的UI元素
```

### Waneella 科幻场景氛围

**风格要素：**
```
光影处理：
- 强烈的光源对比
- 霓虹灯反射
- 体积光效果（用像素模拟）

细节丰富：
- 小尺寸中包含大量细节
- 动态元素（闪烁灯光、移动物体）
- 大气的环境光
```

**应用到游戏：**
```
- 港口场景使用多层光源
- 星球表面细节丰富
- 背景中的小动画（飞船来往、灯光闪烁）
```

## 技术实现建议

### 像素完美渲染

**Web 实现要点：**
```ts
// Canvas 2D：关闭插值，保持像素清晰
const canvas = document.querySelector("canvas")!;
const ctx = canvas.getContext("2d")!;
ctx.imageSmoothingEnabled = false;
```

```css
/* DOM/UI 或 Canvas 外层：像素风渲染提示 */
canvas {
   image-rendering: pixelated;
   image-rendering: crisp-edges;
}
```

```
船只图集（ships_atlas.png）：
```

## 视觉一致性检查清单

**新美术资源审核标准：**
