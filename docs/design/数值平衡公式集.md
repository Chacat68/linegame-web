# 星际贸易商 - 数值平衡公式集

## 系统概述

本文档汇总了《星际贸易商》中所有核心数值计算公式，为开发和平衡调整提供精确的数学模型。所有公式均经过平衡性测试，确保游戏在各个阶段都能保持适当的挑战性和收益性。

本文档补充和扩展了 [15_游戏平衡与数值设计.md](./15_游戏平衡与数值设计.md) 中的内容。

## 1. 贸易核心公式

### 1.1 贸易利润公式

**净利润计算：**
```
净利润 = (卖出价 - 买入价) × 数量 - 运输成本 - 交易税
```

**卖出价计算：**
```
卖出价 = 基础价值 × 供需系数 × 地区系数 × (1 + 谈判技能 × 0.02)

参数说明：
- 基础价值：商品的标准价格
- 供需系数：0.3 ~ 3.0（根据市场供需动态计算）
- 地区系数：0.8 ~ 1.5（不同星球的价格修正）
- 谈判技能：玩家谈判技能等级 (0-10)
- 0.02：每级技能提供2%的价格优势
```

**买入价计算：**
```
买入价 = 基础价值 × 供需系数 × 地区系数 × (1 - 谈判技能 × 0.02)

说明：谈判技能在买入时降低价格，卖出时提高价格
```

**运输成本计算：**
```
运输成本 = 航行距离 × 燃料单价 × (1 / 燃料效率) + 维护费用

参数说明：
- 航行距离：光年（LY）
- 燃料单价（当前实现）：
  基础价格 4 SC/单位（js/data/goods.js），经供需与地区系数后实际约 2-8 SC/单位
  （注：设计目标为 30-80 SC/单位，与当前量级不同；如需调整请同步修改 goods.js 和初始资金）
- 燃料效率：飞船的燃料效率值（单位：光年/单位燃料）
  - 穿梭机（基础）：fuelEff 1.0（当前）
  - 货运飞船：fuelEff 1.2
  - 快速帆船：fuelEff 0.7（越低越省油）
  - 银河巨舰：fuelEff 1.5
- 维护费用：见§4.3 维护费用公式
```

**交易税计算：**
```
交易税 = 卖出总额 × 税率 × (1 - 声望折扣)

参数说明：
- 卖出总额 = 卖出价 × 数量
- 税率：
  - 正规市场：8%
  - 自由港：3%
  - 黑市：0%（但有其他风险成本）
- 声望折扣（当前实现，见 `js/data/playerLevels.js`）：
  - 小有名气（50声望）：2%折扣
  - 声名远扬（150声望）：5%折扣
  - 商业精英（400声望）：8%折扣
  - 银河传奇（800声望）：12%折扣
```

### 1.2 供需系数计算

**供需系数公式：**
```
供需系数 = clamp(基准系数 × (1 + 需求偏差 - 供给偏差), 0.3, 3.0)

说明：
- clamp(x, min, max)：将x限制在[min, max]区间
- 基准系数：1.0（标准供需平衡）
- 最小值0.3：防止价格过低（保护玩家利益）
- 最大值3.0：防止价格过高（保持游戏平衡）
```

**需求偏差计算：**
```
需求偏差 = (当前需求 - 平均需求) / 平均需求

示例：
- 平均需求 = 100单位/天
- 当前需求 = 150单位/天
- 需求偏差 = (150 - 100) / 100 = 0.5（需求高50%，价格上涨）
```

**供给偏差计算：**
```
供给偏差 = (当前库存 - 标准库存) / 标准库存

示例：
- 标准库存 = 200单位
- 当前库存 = 50单位
- 供给偏差 = (50 - 200) / 200 = -0.75（库存不足75%，价格上涨）
```

**综合示例：**
```
基准系数 = 1.0
需求偏差 = 0.5（需求高）
供给偏差 = -0.75（库存低）
供需系数 = clamp(1.0 × (1 + 0.5 - (-0.75)), 0.3, 3.0)
          = clamp(1.0 × 2.25, 0.3, 3.0)
          = 2.25（价格是基础价值的2.25倍）
```

### 1.3 时间效率公式

**时间收益比（每分钟收益）：**
```
时间收益比 = 净利润 / (航行时间 + 交易时间 + 补给时间)

参数说明：
- 净利润：见§1.1
- 航行时间：距离 / 飞船速度（分钟）
- 交易时间：每次交易固定3分钟
- 补给时间：需要补给时固定5分钟
```

**航行时间计算：**
```
航行时间（分钟） = (距离 / 飞船速度) × 60

飞船速度参考：
- 基础货船：0.5 LY/分钟（30 LY/小时）
- 中型货船：0.8 LY/分钟（48 LY/小时）
- 高级货船：1.2 LY/分钟（72 LY/小时）
- 快速飞船：2.0 LY/分钟（120 LY/小时）
```

**路线效率评分：**
```
路线效率评分 = (总利润 / 总时间) × 风险系数

风险系数：
- 安全路线：1.0
- 低风险路线：0.9
- 中风险路线：0.7
- 高风险路线：0.5
- 极高风险路线：0.3
```

## 2. 市场动态公式

### 2.1 玩家行为对价格的影响

**价格冲击公式：**
```
新价格 = 当前价格 × (1 + 价格冲击系数)

价格冲击系数 = -（交易量 / 市场深度）× 弹性系数

参数说明：
- 交易量：玩家本次交易的数量
- 市场深度：星球市场能吸收的交易量（与星球规模相关）
  - 小型殖民地：100单位
  - 中型星球：500单位
  - 大型星球：2000单位
  - 贸易中心：10000单位
- 弹性系数：
  - 基础资源：0.3（价格不敏感）
  - 工业产品：0.5
  - 奢侈品：0.8（价格敏感）
  - 违禁品：1.2（价格高度敏感）
```

**示例计算：**
```
场景：在中型星球（市场深度500）卖出100单位奢侈品

价格冲击系数 = -(100 / 500) × 0.8 = -0.16（价格下降16%）

如果当前价格 = 1000 SC
新价格 = 1000 × (1 - 0.16) = 840 SC

说明：玩家大量卖出会导致价格下跌
```

### 2.2 价格恢复机制

**价格自然恢复公式：**
```
恢复后价格 = 当前价格 + (均衡价格 - 当前价格) × 恢复速率 × 时间间隔

参数说明：
- 均衡价格：基础价值 × 地区系数
- 恢复速率：每小时恢复的比例
  - 高流动性商品（基础资源）：0.2/小时
  - 中流动性商品（工业产品）：0.1/小时
  - 低流动性商品（奢侈品）：0.05/小时
- 时间间隔：距离上次价格冲击的时间（小时）
```

**示例：**
```
均衡价格 = 1000 SC
当前价格 = 840 SC（玩家刚卖出导致下跌）
恢复速率 = 0.1/小时（工业产品）
时间间隔 = 2小时

恢复后价格 = 840 + (1000 - 840) × 0.1 × 2
           = 840 + 160 × 0.2
           = 840 + 32
           = 872 SC

说明：2小时后价格部分恢复
```

### 2.3 随机波动

**日常价格波动：**
```
实际价格 = 基础价格 × (1 + random(-0.05, 0.05))

说明：每天价格有±5%的随机波动，模拟市场自然波动
```

## 3. 经验与等级公式

### 3.1 经验获取

**交易经验公式：**
```
获得经验 = 基础经验 × (1 + 利润率 × 0.5) × 难度系数

参数说明：
- 基础经验：10点
- 利润率：净利润 / 投资额
- 难度系数：
  - 安全路线：1.0
  - 低风险：1.2
  - 中风险：1.5
  - 高风险：2.0
  - 极高风险：3.0
```

**示例：**
```
场景：完成一次高风险交易，利润率30%

获得经验 = 10 × (1 + 0.3 × 0.5) × 2.0
        = 10 × 1.15 × 2.0
        = 23点经验
```

**任务经验公式：**
```
任务经验 = 基础经验 × 任务难度系数

任务难度系数：
- 简单任务：1.0
- 普通任务：2.0
- 困难任务：5.0
- 精英任务：10.0
- 传说任务：20.0
```

### 3.2 升级所需经验

> **实现说明**：当前代码（`js/data/playerLevels.js`）实现为最高 10 级，升级经验固定值如下表，与下方公式不同。公式为长期设计目标，代码值优先。

**当前实现等级经验表（playerLevels.js）：**
```
等级  所需累计经验
1     0
2     100
3     300
4     600
5     1000
6     1600
7     2400
8     3500
9     5000
10    7500（最高等级）
```

**长期设计目标公式（最高 50 级时参考）：**
```
升到N级所需累计经验 = 100 × N^1.5

示例：
- 升到2级：100 × 2^1.5 ≈ 283经验
- 升到5级：100 × 5^1.5 ≈ 1118经验
- 升到10级：100 × 10^1.5 ≈ 3162经验
- 升到20级：100 × 20^1.5 ≈ 8944经验
```

**注**：`15_游戏平衡与数值设计.md` 中的公式（`基础经验 1000 × N^1.5`）与本公式不同，两者均为规划草稿；代码实现以 `playerLevels.js` 为准。



## 4. 飞船运营成本模型

### 4.1 船员薪资

**船员薪资公式：**
```
总船员薪资 = Σ(各船员等级 × 基础薪资 × 技能系数 × 忠诚度折扣)

参数说明：
- 基础薪资：200 SC/周
- 技能系数：
  - 1级船员：1.0
  - 5级船员：2.0
  - 10级船员：5.0
  - 公式：1 + (等级 - 1) × 0.4
- 忠诚度折扣：
  - 忠诚度 < 30：1.5×（要求加薪）
  - 忠诚度 30-70：1.0×（正常薪资）
  - 忠诚度 > 70：0.9×（愿意降薪）
```

**示例计算：**
```
飞船配置：
- 1名10级船长，忠诚度80
- 2名5级船员，忠诚度60
- 3名1级船员，忠诚度50

船长薪资 = 10 × 200 × 5.0 × 0.9 = 9000 SC/周
5级船员薪资 = 5 × 200 × 2.0 × 1.0 × 2人 = 4000 SC/周
1级船员薪资 = 1 × 200 × 1.0 × 1.0 × 3人 = 600 SC/周

总船员薪资 = 9000 + 4000 + 600 = 13600 SC/周
```

### 4.2 燃料消耗

**燃料消耗公式：**
```
燃料消耗（单位） = 航行距离 × (飞船重量 / 燃料效率)

参数说明：
- 航行距离：光年（LY）
- 飞船重量：基础重量 + 货物重量
  - 基础重量：飞船固有重量（吨）
  - 货物重量：当前载货总重量（吨）
- 燃料效率：引擎效率参数
  - 基础引擎：100（低效）
  - 标准引擎：150
  - 高效引擎：200
  - 顶级引擎：300（高效）
```

**燃料成本计算：**
```
燃料成本 = 燃料消耗 × 燃料单价

燃料单价波动：
- 最低价：30 SC/单位（工业星球）
- 平均价：50 SC/单位
- 最高价：80 SC/单位（边缘星球）
```

**示例：**
```
场景：中型货船（基础重量500吨），载货200吨，标准引擎，航行10光年

飞船总重量 = 500 + 200 = 700吨
燃料消耗 = 10 × (700 / 150) = 46.7单位
燃料成本 = 46.7 × 50 = 2335 SC
```

### 4.3 维护费用

**维护费用公式：**
```
维护费用（每周） = 飞船基础价值 × 0.001 × (1 + 磨损度)

参数说明：
- 飞船基础价值：飞船购买价格
- 0.001：每周基础维护率（0.1%）
- 磨损度：0.0 ~ 1.0
  - 0.0：全新状态
  - 0.5：中等磨损
  - 1.0：严重磨损（维护费翻倍）
```

**磨损度累积公式：**
```
磨损度增长 = (航行距离 / 10000) + (战斗次数 × 0.05)

说明：
- 每航行10000光年增加0.1磨损度
- 每次战斗增加0.05磨损度
- 维修可降低磨损度（成本与磨损度成正比）
```

**示例：**
```
飞船价值：500,000 SC
当前磨损度：0.3

维护费用 = 500,000 × 0.001 × (1 + 0.3)
        = 500 × 1.3
        = 650 SC/周
```

### 4.4 保险费用

**保险费公式：**
```
保险费（每周） = 飞船基础价值 × 保险费率 × 风险系数

参数说明：
- 保险费率：0.002（0.2%/周）
- 风险系数：
  - 安全区域活动：0.5
  - 混合区域活动：1.0
  - 危险区域活动：2.0
  - 无法律区域活动：5.0（极高保险费或拒保）
```

**保险赔付：**
```
飞船损毁赔付 = 飞船当前价值 × 赔付比例

赔付比例：
- 基础保险：50%
- 标准保险：75%
- 全额保险：100%（保费×2）
```

### 4.5 日运营成本汇总

**总运营成本公式：**
```
日运营成本 = 船员薪资/7 + 燃料消耗估算 + 维护费用/7 + 保险费/7

示例（中型货船，标准配置）：
- 船员薪资：13600 SC/周 → 1943 SC/日
- 燃料消耗：平均每日航行5光年 → 约1200 SC/日
- 维护费用：650 SC/周 → 93 SC/日
- 保险费：1000 SC/周 → 143 SC/日

日运营成本 = 1943 + 1200 + 93 + 143 = 3379 SC/日
```

## 5. 货币沉淀（Money Sink）机制

### 5.1 渐进性消费

**维护费指数增长：**
```
随着飞船升级，维护费用指数增长

基础货船维护费：500 SC/周
中型货船维护费：2000 SC/周（4倍）
高级货船维护费：8000 SC/周（16倍）
旗舰级维护费：32000 SC/周（64倍）
```

**船员薪资指数增长：**
```
高级船员薪资远高于基础船员

1级船员：200 SC/周
5级船员：1000 SC/周
10级船员：5000 SC/周
传奇船员：20000 SC/周
```

### 5.2 投资性消费

**贸易站建设成本：**
```
贸易站等级      建设成本         每日收益
─────────────────────────────────────
1级（小型）     100,000 SC      500 SC/日
2级（标准）     300,000 SC      2,000 SC/日
3级（大型）     1,000,000 SC    8,000 SC/日
4级（超大型）   3,000,000 SC    30,000 SC/日
5级（贸易中心） 10,000,000 SC   120,000 SC/日

投资回报周期 = 建设成本 / 每日收益
1级：200天，2级：150天，3级：125天，4级：100天，5级：83天
```

**飞船购买成本：**
```
飞船类型        购买成本         货舱容量
────────────────────────────────────
基础货船        80,000 SC       50单位
中型货船        300,000 SC      200单位
高级货船        1,200,000 SC    500单位
重型货船        5,000,000 SC    2,000单位
超级货轮        20,000,000 SC   10,000单位
```

### 5.3 意外性消费

**海盗劫掠损失：**
```
被劫掠损失 = 货物价值 × 损失比例 + 飞船维修费

损失比例：
- 成功逃脱：0%（但消耗燃料和时间）
- 支付赎金：30%（和平解决）
- 战斗失败：60%（货物+飞船损坏）
- 飞船被毁：100%货物 + 保险赔付（如有）
```

**罚款惩罚：**
```
走私被抓罚款 = 违禁品价值 × 3.0 + 基础罚款10,000 SC

重犯惩罚：
- 首次：标准罚款
- 二次：罚款×2
- 三次：罚款×5 + 声望大幅下降
- 四次及以上：监禁（游戏时间损失）+ 罚款×10
```

**突发事件成本：**
```
船员生病：医疗费500-5000 SC
飞船故障：紧急维修5000-50000 SC
货物变质：损失10-50%货物价值
政治庇护请求：支付10000-100000 SC获得声望
```

### 5.4 通胀控制公式

**全局通胀率计算：**
```
全局通胀率 = 玩家总资产 / 标准资产基准

标准资产基准（随等级变化）：
- 5级：100,000 SC
- 10级：500,000 SC
- 20级：5,000,000 SC
- 30级：50,000,000 SC
- 50级：500,000,000 SC
```

**通胀惩罚机制：**
```
当通胀率 > 1.2时触发：

1. NPC价格上调：
   基础商品价格 × (1 + (通胀率 - 1.2) × 0.5)
   
   示例：通胀率1.5，价格上调15%
   
2. 高级商品出现概率降低：
   出现概率 × (1 - (通胀率 - 1.2) × 0.3)
   
   示例：通胀率1.5，出现概率降低9%
   
3. 任务奖励减少：
   任务奖励 × (1 - (通胀率 - 1.2) × 0.2)
   
   示例：通胀率1.5，奖励减少6%
```

**通胀率示例计算：**
```
场景：20级玩家，总资产8,000,000 SC

标准资产基准 = 5,000,000 SC（20级）
通胀率 = 8,000,000 / 5,000,000 = 1.6

触发通胀惩罚：
- 价格上调：(1.6 - 1.2) × 0.5 = 20%
- 稀有品概率：(1.6 - 1.2) × 0.3 = 12%降低
- 任务奖励：(1.6 - 1.2) × 0.2 = 8%减少
```

## 6. 技能系统数值

### 6.1 谈判技能

**价格优势公式：**
```
买入价格折扣 = 谈判技能等级 × 2%
卖出价格加成 = 谈判技能等级 × 2%

最大优势（10级）：±20%
```

**技能升级成本：**
```
升到N级成本 = 5000 × N^2 SC

示例：
- 升到2级：20,000 SC
- 升到5级：125,000 SC
- 升到10级：500,000 SC
```

### 6.2 驾驶技能

**速度加成公式：**
```
飞船速度加成 = 驾驶技能等级 × 3%

最大加成（10级）：+30%速度
```

**燃料效率加成：**
```
燃料效率加成 = 驾驶技能等级 × 2%

最大加成（10级）：-20%燃料消耗
```

### 6.3 战斗技能

**战斗力加成：**
```
战斗力 = 基础战斗力 × (1 + 战斗技能等级 × 0.05)

最大加成（10级）：+50%战斗力
```

**闪避概率：**
```
闪避概率 = 基础闪避率 + 战斗技能等级 × 2%

基础闪避率：10%
最大闪避率（10级）：30%
```

### 6.4 修理技能

**维修成本降低：**
```
维修成本 = 标准成本 × (1 - 修理技能等级 × 0.03)

最大降低（10级）：-30%维修成本
```

**自我修复效率：**
```
自我修复量 = 总损伤 × 修理技能等级 × 5%

最大自修复（10级）：50%损伤可自行修复
```

## 7. 随机事件概率公式

### 7.1 事件触发概率

**基础触发率：**
```
每小时事件触发概率 = 基础概率 × 区域系数 × 运气系数

基础概率：10%/小时
区域系数：
- 安全区：0.5
- 中立区：1.0
- 危险区：2.0
- 无法区：3.0

运气系数（可选技能）：
- 无运气技能：1.0
- 运气+1：1.2
- 运气+5：2.0
```

### 7.2 事件类型分布

**事件类型权重：**
```
总权重 = Σ各事件权重

某事件触发概率 = 该事件权重 / 总权重

示例事件权重分布：
- 普通商机：30（高频）
- 海盗遭遇：15（中频）
- 求救信号：10（中频）
- 稀有发现：5（低频）
- 传奇事件：1（极稀有）

商机触发概率 = 30 / (30+15+10+5+1) = 49.2%
传奇事件概率 = 1 / 61 = 1.6%
```

### 8.1 声望获取

**交易声望公式：**
```
获得声望 = 交易额 / 10000 × 声望系数

声望系数：
- 基础贸易：1.0
- 稀有商品：1.5
- 大宗交易：2.0
- 独家合同：3.0
```

**任务声望公式：**
```
任务声望 = 基础声望 × 任务难度

基础声望：10点
任务难度：1.0 ~ 5.0
```

### 8.2 声望衰减

**声望衰减公式：**
```
每周声望衰减 = 当前声望 × 0.02

说明：每周损失2%声望，鼓励持续交易
最低保底：已获得的勋章不会因衰减而失效
```

## 9. 公式使用示例

### 示例1：计算一次完整交易的利润

**场景设定：**
- 商品：高级电子元件
- 基础价值：1000 SC
- 购买地：工业星球（地区系数0.9，供需系数1.2）
- 销售地：边缘殖民地（地区系数1.3，供需系数2.0）
- 数量：50单位
- 谈判技能：5级
- 距离：15光年
- 飞船：中型货船（燃料效率3.5 LY/单位）
- 燃料单价：50 SC
- 销售地税率：8%，银牌商人（10%折扣）

**计算过程：**

1. 买入价：
```
买入价 = 1000 × 1.2 × 0.9 × (1 - 5 × 0.02)
      = 1000 × 1.2 × 0.9 × 0.9
      = 972 SC/单位
总买入成本 = 972 × 50 = 48,600 SC
```

2. 卖出价：
```
卖出价 = 1000 × 2.0 × 1.3 × (1 + 5 × 0.02)
      = 1000 × 2.0 × 1.3 × 1.1
      = 2,860 SC/单位
总卖出收入 = 2,860 × 50 = 143,000 SC
```

3. 运输成本：
```
燃料消耗 = 15 / 3.5 = 4.29单位
燃料成本 = 4.29 × 50 = 214.5 SC
维护费用 = 忽略（长期平摊）
运输成本 = 214.5 SC
```

4. 交易税：
```
交易税 = 143,000 × 0.08 × (1 - 0.1)
      = 143,000 × 0.072
      = 10,296 SC
```

5. 净利润：
```
净利润 = 143,000 - 48,600 - 214.5 - 10,296
      = 83,889.5 SC
利润率 = 83,889.5 / 48,600 = 172.6%
```

**结论：**这是一条高利润路线，符合高风险高回报的设计。

## 10. 平衡调整指南

### 10.1 调整原则

1. **保持核心公式结构不变**，优先调整系数参数
2. **渐进式调整**，每次修改单个参数，测试影响
3. **数据驱动**，基于玩家数据分析调整
4. **关注极值**，确保边界情况不会破坏平衡

### 10.2 常见平衡问题及解决方案

**问题1：玩家资金增长过快**
- 降低高利润商品的供需系数上限（3.0 → 2.5）
- 提高交易税率（8% → 10%）
- 增加通胀惩罚力度

**问题2：玩家资金增长过慢**
- 提高任务奖励基数
- 降低飞船运营成本
- 增加低风险路线的利润率下限

**问题3：后期内容缺乏吸引力**
- 调整高级飞船的效率加成
- 增加大额投资的回报率
- 引入新的Money Sink项目

### 10.3 测试检查清单

每次调整公式后检查：
- [ ] 新手能否在1小时内完成首次盈利循环
- [ ] 中期玩家（10级）能否在合理时间内购买第二艘飞船
- [ ] 后期玩家（30级+）是否仍有明确的进步目标
- [ ] 各种玩法路线（和平/走私/战斗）收益是否平衡
- [ ] Money Sink是否有效防止经济膨胀
