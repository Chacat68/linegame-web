# 星际贸易商 - 新手引导设计

## 系统概述

新手引导系统是玩家进入游戏世界的第一道门槛，良好的教程设计能够有效降低学习曲线，提高玩家留存率。《星际贸易商》的新手引导采用渐进式教学，将复杂的游戏系统分解为4个阶段，每个阶段专注于特定的核心机制，确保玩家在获得成就感的同时逐步掌握游戏玩法。

## 新手引导四阶段设计

## 教学原则

### 1. 渐进式解锁

**信息分层：**
```
阶段1：只展示最基础的UI（星图、市场）
阶段2：解锁交易详情（利润计算、经验系统）
阶段3：解锁战略工具（路线规划、燃料管理）
阶段4：解锁风险管理（随机事件、保险、维修）
```

**避免信息过载：**
- 每个阶段只引入2-3个新概念
- 新功能解锁时用动画和音效强调
- 复杂系统（如派系、舰队）在教程后才出现

### 2. 即时反馈

**每个操作都有明确反馈：**
- 购买商品：货舱UI更新 + 音效
- 航行：进度条 + 星空动画
- 出售商品：金币飞入动画 + 利润弹窗
- 升级：全屏特效 + 解锁清单

## 数据埋点与优化

### 关键指标监测

**教程完成率：**
```
阶段1完成率：目标 >95%
阶段2完成率：目标 >85%

阶段4完成率：目标 >60%

```

**卡点识别：**
```
追踪每个步骤的停留时间：
- 超过5分钟未操作：触发提示
- 超过3次错误操作：简化当前步骤
- 重复打开帮助：说明说明不够清晰
```
