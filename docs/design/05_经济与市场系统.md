# 星际贸易商 - 经济系统设计

## 系统概述

经济系统是《星际贸易商》的核心机制之一，通过模拟真实的经济运行规律，为玩家提供深度的经济策略体验。本系统包含宏观经济、微观经济、产业链和金融系统等多个层面。

## 宏观经济系统

### 经济周期

1. **繁荣期**
   - 特征：高需求、高价格、高利润
   - 持续时间：2-4个游戏月
   - 投资机会：扩张生产、囤积货物
   - 风险：泡沫形成、过度投资

2. **稳定期**
   - 特征：供需平衡、价格稳定
   - 持续时间：3-6个游戏月
   - 投资机会：优化运营、建立储备
   - 风险：竞争加剧、利润下降

3. **衰退期**
   - 特征：需求下降、价格下跌
   - 持续时间：2-4个游戏月
   - 投资机会：收购资产、降低成本
   - 风险：资金链断裂、资产贬值

4. **萧条期**
   - 特征：需求极低、价格暴跌
   - 持续时间：1-3个游戏月
   - 投资机会：抄底资产、战略储备
   - 风险：市场崩溃、信用危机
